2. Web server là gì ?

Web server là một phần mềm máy tính hoạt động trên 1 phần cứng và chấp nhật các requests thông qua giao thức HTTP để phân phối các trang webs, biến thể an toàn hơn của HTTP là HTTPS.

Để publish 1 website thì bạn cần hoặc 1 web server tĩnh (chỉ bao gồm 1 máy tính và 1 HTTP server, gọi là tĩnh vì server gửi những files được hosted trên server đến browers) hoặc động (có thêm 1 số phần mềm như ứng dụng server và 1 database, gọi là động vì ứng dụng server sẽ cập nhật hosted files trước khi gửi đến browser thông qua HTTP server).

3.Những web servers phổ biến nhất

  1. Apache HTTP Server: Được phát trển bởi Apache Software Foundation, free và mã nguồn mở có thể hoạt động trên cả Windows MacOS Unix Linux Solaris và các OS khác.
  2. Microsoft Internet Information Services (IIS): Phát triển bởi Microsoft cho các nền tảng của Microsoft, không open-sourced nhưng được sử dụng rộng rãi.
  3. Nginx: Open-sourced, phổ biến vì nó sử dụng ít tài nguyên cũng như khả năng scale mạnh, có thể xử lý nhiều sessions đồng thời nhờ vào kiến trúc hướng sự kiện (event-driven architecture), ngoài ra thì Nginx còn được sử dụng như 1 proxy server, load balancer.
  4. Lighttpd: free và chạy trên FREEBSD OS, nhanh và bảo mật, tốn ít tài nguyên CPU.
  5. Sun Java System Web Server.
